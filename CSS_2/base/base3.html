Блок навигации


  • Ширина 15см
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточный, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере
Блок навигации наполните контентом достаточный, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере.

Статья1


Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равный 2см, при помощи margin-bottom

Статья2


Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равный 2см, при помощи margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной стать для этой картинки
